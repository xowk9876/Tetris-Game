# 🎵 테트리스 게임 사운드 파일

이 디렉토리에는 테트리스 게임에 사용되는 사운드 파일들이 들어갑니다.

## 사운드 파일

모든 사운드 파일은 **선택적**입니다. 파일이 없어도 게임이 정상 작동합니다.

### 실제 사용되는 사운드

| 파일명 | 설명 | 재생 시점 |
|--------|------|----------|
| `bgm.mp3` 또는 `bgm.wav` | 배경음악 | 게임 시작 시 루프 재생 |
| `drop.wav` 또는 `drop.mp3` | 블록 낙하 | 블록 고정 시 |
| `game_over.wav` 또는 `game_over.mp3` | 게임 오버 | 게임 종료 시 |
| `tetris.wav` 또는 `tetris.mp3` | 테트리스 | 4줄 제거 시 (또는 line_clear 없을 때) |

### 선택 사운드

| 파일명 | 설명 | 재생 시점 |
|--------|------|----------|
| `move.wav` 또는 `move.mp3` | 블록 이동 | 좌우 이동 시 |
| `rotate.wav` 또는 `rotate.mp3` | 블록 회전 | 회전 시 |
| `line_clear.wav` 또는 `line_clear.mp3` | 줄 제거 | 1~3줄 제거 시 (없으면 tetris 사용) |
| `hold.wav` 또는 `hold.mp3` | 홀드 | 홀드 사용 시 |
| `combo.wav` 또는 `combo.mp3` | 콤보 | 콤보 2 이상 시 |

## 파일 형식

- **형식**: MP3 또는 WAV
- **효과음**: 50KB 이하 권장
- **배경음악**: 2MB 이하, 루프 가능한 파일 권장

## 사용 방법

1. 사운드 파일을 이 디렉토리에 추가
2. 게임이 자동으로 로드하고 재생합니다
3. 파일이 없어도 게임은 정상 작동합니다
